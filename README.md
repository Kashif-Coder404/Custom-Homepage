# Custom Dashboard (Browser Homepage & Live Wallpaper)

A highly personalized, feature-rich custom dashboard designed to be used as your browser's new tab page or as a live desktop wallpaper (using apps like [Lively Wallpaper](https://rocksdanister.github.io/lively/)). 

This dashboard is built from scratch to combine powerful productivity tools, universal AI search capabilities, and personal habit trackers all in one customizable, modern interface.

## ✨ Key Features

### 🎨 Complete Customization
- **Live Video & Wallpaper Uploads:** Fully personalize your workspace by uploading your own video loops or high-resolution images as the background.
- **Floating Modular Widgets:** Drag, drop, and rearrange widgets anywhere on your screen to create a layout that fits your workflow perfectly.
- **Modern Glassmorphism UI:** A sleek, premium aesthetic with ambient background orbs.

### 🧠 Universal AI & Search Hub
- **Instant Search Toggle:** Switch your search queries instantly between Google, ChatGPT, Claude, Gemini, Perplexity, YouTube, and Wikipedia from a single search bar. 
- **Categorized Quick Links:** Fast access to top Developer tools (GitHub, MDN, LeetCode), AI tools, Design platforms, and Social Media.

### ✅ Productivity Built-in
- **Task Manager:** Keep track of your daily to-dos with a visual progress ring.
- **Quick Notes:** A distraction-free notepad to jot down thoughts on the fly.
- **Clock & Date:** A minimalist widget to keep you on schedule.

### 🕌 Personal Habits & Faith Tracker
- **Salah Tracker Heatmap:** Log your daily prayers and visualize your consistency over time.
- **Daily Reflections (Dua):** Learn new daily reflections with English translations and transcriptions.
- **Tasbeeh Counter:** A functional counter to stay spiritually grounded.

## 🚀 How to Use It

### As a Browser Homepage
You can set this project as your default new tab page in your browser (like Chrome, Brave, or Edge) by using a custom new tab extension and pointing it to the local `index.html` file, or by hosting it on GitHub Pages.

### As a Desktop Live Wallpaper
To use this interactive dashboard as your actual Windows desktop wallpaper:
1. Download and install [Lively Wallpaper](https://rocksdanister.github.io/lively/).
2. Drag and drop the `index.html` file (or the entire project folder) into the Lively Wallpaper library.
3. Apply it! The widgets and video uploads will work directly on your desktop.
